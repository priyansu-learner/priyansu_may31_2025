The Agrichain_TC - Sheet1.csv file contains the testcases of the given website.

The Selenium_Test folder contains the high level Automation framework using Selenium.
All automation related test files are under src/test/java/com/example.

The browserinitialization folder contains the code to intialize the browser.
The pageObjects folder contains the locators and action functions of each page separately.
The resources folder contains the external resources like text,input/output values.
The test folder contains the testfile.
The utils folder contains helper methods.
